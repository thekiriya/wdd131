Overview

Purpose

A site for locally storing books. You're also able to choose custom book covers and input the description of the book as well as select a rating out of 5.

Audience

Book Worms or anyone looking to get into tracking their books virtually.

Dynamic elements

Objects: Each books is an object Arrays: Collection of Bookshelf DOM manipulation: load book data automatically Conditionals: displaying rating
